# Restaurant-website
🍽️ A modern, responsive fine-dining restaurant website featuring an elegant UI and a real-time reservation system powered by Firebase.
Gourmet Gateway 🍽️

Gourmet Gateway is a modern, single-page website for a fine-dining restaurant. It features an elegant, responsive design, a featured menu showcase, and a real-time reservation system powered by Firebase.

✨ Features

Elegant UI/UX: A sophisticated design using a warm, terra cotta color palette tailored for fine dining.

Fully Responsive: Built with Tailwind CSS to ensure a seamless experience across desktop, tablet, and mobile devices.

Real-Time Reservations: Users can book a table using the integrated form.

Live Booking Ledger: A real-time feed that displays recent reservations fetched instantly from a Firebase Firestore database.

Smooth Animations: Includes subtle CSS animations and transitions for a polished feel.

🛠️ Technologies Used

Frontend: HTML5, JavaScript (ES6+), Tailwind CSS (via CDN)

Backend / Database: Firebase (v11.6.1)

Firebase Authentication (Anonymous/Custom Token)

Cloud Firestore (Real-time NoSQL database)

Typography & Icons: Google Fonts (Playfair Display, Inter), FontAwesome (v6.4.0)
